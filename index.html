Reglas

Durante 24 horas los participantes deberán construir un juego basado en HTML5 y JavaScript cuyas reglas y requerimientos serán dados a conocer el día de la competencia.

El código construido será enviado a nuestros servidores para hacer el despliegue y evaluar la calidad del trabajo.

Premios

  • $500 USD al equipo ganador
  • $150 USD al segundo equipo

Jurados

Tenemos el gusto de contar con la colaboración de:

  • John Acosta (@John_Acosta): Fundador y organizador de BogotaJS, la comunidad más grande de desarrolladores en Javascript de Colombia
  • Juan Pablo Buritica (@buritica): Fundador y organizador del BogotaConf2011, desarrollador back y front-end
  • David Avellaneda (@Davsket): cofundador de Monoku y programador Javascript

Organización

  • El evento sera online, existe la posibilidad de disponer de un sitio para que los participantes en la ciudad de Bogotá puedan trabajar, tomar cerveza y comer pizza.
  • Convocatoria abierta para cualquier interesado en Colombia.
  • Pueden participar equipos de 1 hasta n personas.

Condiciones

  • Aplicación únicamente en JavaScript y HTML5 (no se vale Unity, Flash u otros parecidos, si se vale Coffescript o algún otro compilador/generador de JavaScript).
  • Para efectos de la competencia todo el código que se envíe para ser evaluado quedara disponible en Github para que cualquiera pueda verlo, luego de esto los equipos pueden disponer de el.
  • La información de contacto de todos los participantes podrá ser usada por la organización o los patrocinadores únicamente para informar de otros eventos o contacto laboral.
  • La competencia se realizará a través de Github, por este motivo es importante que antes de empezar la competencia los equipos aprendan a usar git y tengan una cuenta válida de Github
  • Además de evaluar que se cumplan todos los requisitos, la organización evaluará conceptos como estructura del código, programación orientada a objetos, algoritmos implementados y otros elementos técnicos que los jurados consideren relevantes.
